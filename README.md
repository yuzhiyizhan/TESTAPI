网上找的一些免费接口,用unittest搭建自动化接口测试框架

1.环境

ubuntu18.04LST

windowns 10

2.运行

python3 runall.py

3.配置

配置文件在config有详细说明

4.添加测试用例

在files文件夹放入以.xls为后缀的接口测试文档,文档架构参考APi.xls,如需更改,请自行修改

在testapi.py添加如下方法

def test_03(self):
    self.Request()

意思是执行文档里第三条测试用例

可以在Request里面填入用例的说明
